What DevilTongues Does

DevilTongues is a research and educational analytics platform that detects synthetic bond arbitrage opportunities via real-time put-call parity analysis. It connects to LSEG Refinitiv market data to retrieve equity quotes and options chains, then computes implied risk-free rates and highlights conversion and reverse-conversion signals.

Key idea: Synthetic bond (or synthetic stock) positions built from options can be mispriced relative to the underlying. When the implied rate deviates from a benchmark beyond a threshold, arbitrage opportunities may exist.

Key Features

  • Real-time market data via LSEG Refinitiv Data API (spot + options chain).
  • Implied rate computation from put-call parity across strikes and expiries.
  • Signal detection with configurable thresholds and ranking.
  • Strategy breakdown with positions, risks, and cost modeling.
  • Visualization including 3D surface plots for term structures.

Data Sources

The platform uses enterprise-grade market data infrastructure provided by LSEG Refinitiv. Typical fields include spot quotes (e.g., TR.PriceClose) and options pricing fields (e.g., CF_BID, CF_ASK, CF_LAST).
