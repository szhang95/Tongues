Prerequisites

  • Python 3.12
  • LSEG Refinitiv Data API credentials
  • LSEG Workspace or Eikon desktop app running and signed in

Installation

  1. Clone the repository
git clone https://github.com/deviltongues/Tongues.git cd Tongues
  1. Create and activate a virtual environment
python -m venv .venv # macOS/Linux source .venv/bin/activate # Windows # .venv\Scripts\activate
  1. Install dependencies
pip install -e .

Authentication Notes

The application typically uses desktop authentication through Workspace/Eikon. Keep the desktop application running while using DevilTongues and confirm that API access is enabled for your account.

Launch

shiny run app.py

By default, the app starts at http://127.0.0.1:8000.
