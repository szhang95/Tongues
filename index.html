What is DevilTongues?

A sophisticated financial analytics platform that identifies arbitrage opportunities in synthetic bonds through real-time put-call parity analysis

Core Concept: Synthetic bonds are created by combining options positions to replicate stock payoffs. When these synthetics are mispriced relative to actual stock, arbitrage opportunities emerge. DevilTongues automatically detects these opportunities in real-time.

How It Works

DevilTongues exploits deviations from put-call parity to identify arbitrage opportunities

Put-Call Parity Foundation

C - P = S - K √ó e-rT

Where: C = Call price, P = Put price, S = Spot price, K = Strike, r = Risk-free rate, T = Time to expiry

Reverse Conversion

When: Implied rate > Benchmark rate

Action: Sell synthetic (Sell Call + Buy Put) + Buy Stock

Profit: Collect premium from overpriced synthetic

Conversion

When: Implied rate < Benchmark rate

Action: Buy synthetic (Buy Call + Sell Put) + Short Stock

Profit: Buy cheap synthetic, earn interest on short proceeds

Quick Start Guide

Get DevilTongues running locally in minutes

1

Clone Repository

Download the source code from GitHub

git clone https://github.com/deviltongues/Tongues.git cd Tongues
2

Setup Environment

Create virtual environment and install dependencies

python -m venv .venv # On Windows: .venv\Scripts\activate # On macOS/Linux: source .venv/bin/activate pip install -e .
3

Configure LSEG Access

Ensure LSEG Workspace or Eikon desktop application is running and you're logged in with valid credentials

4

Launch Application

Start the Shiny application

shiny run app.py # Open browser to http://127.0.0.1:8000
