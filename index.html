Overview

DevilTongues is a sophisticated financial analytics platform that identifies arbitrage opportunities in synthetic bonds through real-time put-call parity analysis. By connecting to LSEG Refinitiv's market data, the platform calculates implied risk-free rates from options pricing and flags profitable conversion and reverse conversion strategies.

What are Synthetic Bonds?
Synthetic bonds are created by combining options positions to replicate the payoff of holding (or shorting) the underlying stock. When these synthetic positions are mispriced relative to the actual stock, arbitrage opportunities emerge.

This is a research and educational tool. The platform provides analysis and visualization only - it does not execute trades automatically. Users must run the application locally with their own LSEG credentials.

Strategy Explanation

Put-Call Parity Foundation

The platform exploits deviations from put-call parity, a fundamental no-arbitrage relationship in options pricing:

C - P = S - K √ó e-rT

Where:

  • C = Call option price
  • P = Put option price (same strike & expiry)
  • S = Spot price of underlying
  • K = Strike price
  • r = Risk-free interest rate
  • T = Time to expiration (years)

Arbitrage Strategies

Reverse Conversion (Implied Rate > Benchmark)

When: The synthetic short stock is overpriced relative to actual stock

Positions:

  1. SELL Call option at strike K
  2. BUY Put option at strike K
  3. BUY underlying stock

Profit Source: Collect premium from selling the expensive synthetic (short call + long put combination).

Conversion (Implied Rate < Benchmark)

When: The synthetic long stock is underpriced relative to actual stock

Positions:

  1. BUY Call option at strike K
  2. SELL Put option at strike K
  3. SHORT underlying stock

Profit Source: Buy cheap synthetic (long call + short put combination), earn interest on short proceeds.

‚ö†Ô∏è Important Risks:
  • Execution risk across 3 simultaneous legs
  • Pin risk at expiration if stock ‚âà strike
  • Early assignment on American-style short options
  • Dividend payments affecting put-call parity
  • Transaction costs may erode thin arbitrage margins
  • Margin requirements for short stock positions

How to Run Locally

üìã Prerequisites Required:
  • Python 3.12 (3.13 not yet supported by all dependencies)
  • LSEG Refinitiv Data API credentials
  • LSEG Workspace or Eikon desktop application running and logged in

Installation Steps

1Clone the Repository

git clone https://github.com/deviltongues/Tongues.git cd Tongues

2Create Virtual Environment

python -m venv .venv # Activate virtual environment # On Windows: .venv\Scripts\activate # On macOS/Linux: source .venv/bin/activate

3Install Dependencies

pip install -e .

This installs all required packages: shiny, refinitiv.data, pandas, numpy, plotly, scipy

4Configure LSEG Authentication

The application uses LSEG Workspace desktop authentication. Before running:
  • Launch LSEG Workspace or Eikon desktop application
  • Log in with your valid credentials
  • Verify API access is enabled in your account
  • Keep the desktop application running while using DevilTongues

5Launch Application

shiny run app.py

The application will start on http://127.0.0.1:8000 by default. Open this URL in your web browser.

First-Time Usage

  1. Fetch Spot Price: Select an underlying (e.g., MSFT.O) from the dropdown and click "Fetch Spot Price" to retrieve the current stock price.
  2. Configure Options Chain: Set Min/Max Strike prices (e.g., 400-500), Min/Max Expiry dates, and Max Contracts. Click "Scan Options" to retrieve the options chain.
  3. Analyze Arbitrage: Set Risk-Free Rate (e.g., 5.0%) and Arbitrage Threshold (e.g., 0.5%). Click "Analyze Arbitrage" to detect opportunities.
  4. Review Results: Navigate to the "Arbitrage Signals" tab to view detected opportunities. Click on any row to see detailed strategy breakdowns.
  5. Calculate Costs: Use the "Execution Calculator" tab to model real-world costs with your position size, commissions, and expected slippage.
